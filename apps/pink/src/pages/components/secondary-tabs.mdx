---
title: Secondary Tabs
description: Secondary tabs allows users to switch between multiple sections of content within a single section/container.
---

| Class                 | Type            |                                                 |
| --------------------- | --------------- | ----------------------------------------------- |
| `secondary-tabs`      | Secondary Tabs  | A class representing secondary tabs             |

## Components

Secondary Tabs consist of two components:

| Class                   | Type   |             |
| ----------------------- | ------ | ----------- |
| `secondary-tabs-item`   | Item   |             |
| `secondary-tabs-button` | Button |             |

## Stretch Tabs

Tabs stretch to fill all line:

| Class                   | Behave   |
| ----------------------- | -------- |
| `is-stretch`            | Stretch  |
