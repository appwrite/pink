---
title: Output
description: Interactive text outputs allow users to interact with the displayed text. Copying and hiding are the two most common actions in this element.
---

| Class                      | Type    |                                                      |
| -------------------------- | ------- | ---------------------------------------------------- |
| `interactive-text-output`  | Output  | A class representing an interactive text output      |

## Components

Interactive text output contains buttons:

| Class                            | Type             |                                                                                            |
| -------------------------------- | ---------------- | ------------------------------------------------------------------------------------------ |
| `interactive-text-output-buttons`| Button/s Wrapper |                                                                                            |
| `interactive-text-output-button` | Button           | Buttons can be hidden (will appear on hover) or displayed next to the content.             |

<Preview center>

    <div class="interactive-text-output-buttons">
